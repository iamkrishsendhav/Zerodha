🚀 Zerodha Clone (Full Stack MERN App)

A production-ready full-stack Zerodha clone with authentication, OTP verification, and a responsive trading dashboard.

🌐 Live Demo

👉 https://zerodha-frontend-final-eta.vercel.app/

🧠 Project Overview

This project replicates core functionalities of Zerodha, focusing on real-world authentication flow, dashboard management, and deployment practices.

🛠️ Tech Stack

Frontend

- React.js
- React Router DOM
- Axios
- Bootstrap & Custom CSS

Backend

- Node.js
- Express.js
- JWT Authentication
- Bcrypt (Password Hashing)

Database

- MongoDB Atlas
- Mongoose

Email System

- Nodemailer (OTP Verification)

Deployment

- Frontend & Dashboard → Vercel
- Backend → Render

---

✨ Key Features

- 🔐 Secure Authentication (JWT)
- 📧 Email OTP Verification (Signup & Forgot Password)
- 🔑 Password Reset Flow
- 📊 Dashboard (Holdings & Positions)
- 🚫 Protected Routes
- 🎨 Clean UI (Zerodha Inspired)
- 🌐 Fully Deployed Application

📚 What I Learned

- Real-world authentication system design
- OTP-based email verification
- Backend deployment (Render)
